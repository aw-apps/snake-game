# 🐍 貪食蛇遊戲

復刻版貪食蛇網頁遊戲，使用純 HTML/CSS/JS 開發，無任何框架依賴。

## 功能

- 20×20 格遊戲畫面，Canvas 渲染
- 鍵盤控制（方向鍵 / WASD）、空白鍵暫停
- 行動裝置觸控方向按鈕
- 即時分數顯示與遊戲狀態（Running / Paused / Game Over）
- 暫停 / 繼續 / 重新開始按鈕

## 遊玩

線上遊玩：[https://aw-apps.github.io/snake-game/](https://aw-apps.github.io/snake-game/)

## 本機開發

```bash
# 任意靜態伺服器即可，例如
npx serve .
```

## 技術

- HTML5 Canvas
- 純 CSS（無框架）
- ES Modules（無打包工具）
- GitHub Pages 部署
